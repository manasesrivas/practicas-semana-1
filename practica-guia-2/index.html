Que es html?

Html es un lenguaje de marcado que se utiliza para estructurar contenido en la web utilizando el protocolo HTTP

Que es HTTP?

HTTP es un protocolo de comunicacion utilizado para la web. se encarga de la comunicacion entre clientes y servidores.
